# Host, Switch, Network

Date: 2023년 10월 10일

Host : 네트워크에 연결된 컴퓨터

- 네트워크 이용 주체 : End Point → Peer, Server, Client
- 네트워크 구성 주체 : Switch → Router(경로 찾는 스위치), FW(보안 설정하는 스위치), IPS

Internet

- Router와 DNS의 집합체

Switch의 분류

- HTTP로 스위치 : L7 스위치
- Port 번호로 스위치 : L4 스위치
- IP 주소로 스위치 : L3 스위치 → 가장 대표적인 것 `라우터`! 그리고 `공유기`
- MAC 주소로 스위치 : L2 스위치
